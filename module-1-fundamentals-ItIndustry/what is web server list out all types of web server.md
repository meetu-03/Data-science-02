  # Web Servers

A **Web Server** is software (and sometimes hardware) that receives HTTP/HTTPS requests from clients (web browsers) and delivers web pages, files, APIs, and other web content.

## Types of Web Servers

### 1. Static Web Servers
Serve static content such as HTML, CSS, JavaScript, images, and files without server-side processing.

Examples:
- Apache HTTP Server
- Nginx
- Lighttpd

### 2. Dynamic Web Servers
Generate content dynamically using server-side programming languages and databases.

Examples:
- Apache + PHP
- Nginx + PHP-FPM
- Microsoft IIS + ASP.NET
- Node.js Server

### 3. Reverse Proxy Servers
Receive client requests and forward them to backend servers.

Examples:
- Nginx
- HAProxy
- Traefik
- Envoy Proxy

### 4. Application Servers
Host and execute web applications and business logic.

Examples:
- Apache Tomcat
- WildFly (JBoss)
- GlassFish
- Jetty

### 5. Cloud-Based Web Servers
Managed web servers hosted in cloud environments.

Examples:
- AWS Elastic Beanstalk
- Google App Engine
- Azure App Service

### 6. Lightweight Web Servers
Designed for low-resource systems and embedded devices.

Examples:
- Lighttpd
- Cherokee
- MiniHTTPD
- Boa

### 7. Development Web Servers
Used during software development and testing.

Examples:
- Python HTTP Server
- Node.js HTTP Server
- PHP Built-in Server
- Django Development Server

## Popular Web Server Software

### Apache HTTP Server
- Open-source
- Cross-platform
- Highly configurable
- Supports modules and extensions

### Nginx
- High performance
- Reverse proxy support
- Load balancing
- Low memory usage

### Microsoft IIS
- Developed by Microsoft
- Runs on Windows Server
- Supports ASP.NET applications

### LiteSpeed
- Commercial web server
- High performance
- Apache-compatible

### Lighttpd
- Lightweight
- Suitable for low-resource environments

### Caddy
- Automatic HTTPS support
- Easy configuration
- Modern web server

### Apache Tomcat
- Java application server
- Runs Java Servlets and JSP applications

### Jetty
- Lightweight Java web server
- Embeddable in applications

## Common Web Server Features

- HTTP/HTTPS Support
- SSL/TLS Encryption
- Virtual Hosting
- Load Balancing
- Reverse Proxy
- Caching
- URL Rewriting
- Authentication & Authorization
- Logging & Monitoring
