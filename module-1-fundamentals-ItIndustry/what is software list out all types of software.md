# What is Software?

## Definition

Software is a collection of programs, instructions, and data that tells a computer how to perform specific tasks.

Unlike hardware, software cannot be touched physically. It works with hardware to perform operations and solve problems.

## Examples of Software

- Microsoft Word
- Microsoft Excel
- Google Chrome
- Mozilla Firefox
- VLC Media Player
- Adobe Photoshop
- WhatsApp
- Zoom
- Power BI
- Visual Studio Code

## Software vs Hardware

| Software | Hardware |
|-----------|-----------|
| Intangible (cannot be touched) | Physical components |
| Instructions and programs | Devices and equipment |
| Can be updated easily | Requires physical replacement |

## Conclusion

Software is a set of programs and instructions that enables a computer to perform tasks. It is broadly classified into System Software, Application Software, and Programming Software.
