# What is Software?

## Definition

Software is a collection of programs, instructions, and data that tells a computer how to perform specific tasks.

Unlike hardware, software cannot be touched physically. It works with hardware to perform operations and solve problems.

## Examples of Software

- Microsoft Excel
- Google Chrome
- Mozilla Firefox
- VLC Media Player
- Adobe Photoshop
- WhatsApp
- Zoom
- Power BI
- Visual Studio Code
