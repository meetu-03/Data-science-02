# What is CSV?

## Definition
CSV (Comma-Separated Values) is a simple file format used to store tabular data such as spreadsheets or databases.

In a CSV file, each line represents a row, and values are separated by commas (,).

## Example of a CSV File

```csv
ID,Name,Age,City
1,Rahul,21,Rajkot
2,Priya,22,Ahmedabad
3,Amit,20,Surat
4,Neha,23,Vadodara
```

## How the Data Appears in a Table

| ID | Name  | Age | City      |
|----|-------|-----|-----------|
| 1  | Rahul | 21  | Rajkot    |
| 2  | Priya | 22  | Ahmedabad |
| 3  | Amit  | 20  | Surat     |
| 4  | Neha  | 23  | Vadodara  |

## Uses of CSV

- Storing spreadsheet data
- Importing and exporting data between applications
- Database backups
- Data analysis using Python, Excel, and SQL tools

## Advantages

- Simple and lightweight
- Easy to read and edit
- Supported by Excel, Google Sheets, and databases
- Easy to exchange data between systems

## Conclusion

CSV is a text-based file format used to store tabular data where values are separated by commas. It is widely used for data storage, sharing, and analysis.
